This public API is developed to help patients who need emergency blood transfusion. Blood bank services in Yangon have many challenges of meeting various blood transfusion requirements. This API connected to a database which holds the details of blood donors in Yangon. Users can search for a required blood type in the database and contact the donor directly. Also, developers can use this API to integrate with their applications of various types. The API is created using Node JS and MongoDB database. If you're a blood donor who wants to provide the details, please email to: andyweb3@protonmail.com
